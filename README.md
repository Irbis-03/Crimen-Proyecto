# Crimen-Proyecto
Este código crea una aplicación web interactiva usando Streamlit para analizar datos de crímenes en Puerto Rico entre 2013 y 2016. Carga un dataset desde internet, limpia y asigna nombres a las columnas, y permite al usuario filtrar la información por área policiaca, tipo de delito, día de la semana y horario (AM/PM). A partir de los datos filtrados, calcula métricas clave como la cantidad de incidentes, el delito más frecuente y el área con más casos. Además, asigna un índice de gravedad a cada delito y presenta los resultados en un mapa geográfico interactivo y una gráfica de barras que muestra la distribución de delitos.
